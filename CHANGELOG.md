# v1.0.0

- Initial release.
- Added managed farm regions through WorldGuard.
  - Protected immature crops from accidental breaking.
  - Automatically harvested and replanted fully grown crops.
  - Protected farmland from trampling.
- Added `/fm region add|remove|list` for choosing managed farm regions.
  - Region names can be written as `world:region`.
- Added configurable crop behavior.
  - Server owners can choose which supported crops FarmManager handles.
  - Creative mode behavior, sneaking bypass, inventory delivery, and overflow handling can be adjusted.
- Added configurable composter behavior.
  - Server owners can add custom compostable items or override compost chances for existing items.
  - Custom composter rules apply to both player use and hopper input.
- Added English and Korean messages.
